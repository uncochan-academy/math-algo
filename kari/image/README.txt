====================================
  顔認識プログラム - 使い方
====================================

【概要】
カメラから顔を検出・認識し、名前を学習・表示するプログラム

【必要環境】
- Windowsコンピュータ
- Webカメラ

【起動方法】
FaceRecognition.exe をダブルクリック

【基本的な使い方】

1. 顔を登録する
   nキー → 名前を入力 → Enterキー
   → カメラに顔を向ける（5回自動撮影）

2. 学習する
   tキー → 学習完了まで待つ

3. 認識モードをONにする
   rキー → 顔に名前が表示される

4. 終了する
   qキーまたはEscキー

【全キー操作】
  n : 新しい人を登録（名前入力→顔を5回撮影）
  t : 学習を実行
  s : データを保存
  r : 認識モードON/OFF切り替え
  k : 追跡モードON/OFF切り替え
  c : すべてのデータをクリア
  +/- : 認識状態の維持時間を調整
  q/ESC : 終了

【追跡モード】
kキーでON/OFFを切り替え
- 緑の枠：顔を検出中
- 黄色の枠：動きを追跡中
- 赤の枠：位置を予測中

【データの保存場所】
FaceRecognition.exeと同じフォルダに以下のファイルが作成されます：
- face_data.pkl : 登録した顔データ
- face_recognizer.yml : 学習済みモデル

【ヒント】
- 複数の人を登録する場合、それぞれnキーで登録してからtキーで学習
- 認識精度を上げるには、異なる角度や表情で複数回登録
- データを消したい場合はcキーでクリア

【トラブルシューティング】
- カメラが起動しない → 他のアプリでカメラを使用していないか確認
- 認識されない → 照明を明るくする、カメラに近づく
- 名前が間違っている → cキーでクリアして再登録
